A FoundryVTT Module to greatly simplify token creation from images. It adds a button to the Token Controls Tools and to the Token HUD to quickly select an image for a token and give it an appropriate border. 

Tested and released under FoundryVTT 14.

The included Token Borders are originally created by https://github.com/RPTools/TokenTool and [licensed under APGL-3.0](https://github.com/RPTools/TokenTool/blob/main/LICENSE.md). The .png versions used here have been exported by https://github.com/jcolson/token_frames and [licensed under MIT](https://github.com/jcolson/token_frames?tab=MIT-1-ov-file#readme).
